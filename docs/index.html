Características clave

  • Panel web para definir entornos y cuentas cloud.
  • Integración con múltiples proveedores: AWS / Azure / GCP / VMware / Proxmox.
  • Despliegue de:
    • Prometheus + Grafana
    • Apache SkyWalking (extensible)
    • ELK / OpenSearch
    • Wazuh (extensible)
    • ArgoCD
  • Etiquetado de recursos por entorno, proveedor y cuenta.

Cómo arrancar el panel

git clone https://github.com/TU-USUARIO/hybrid-sre-platform.git
cd hybrid-sre-platform
cp .env.example .env
docker compose up -d --build

# Panel: http://localhost:8000
